# 🎹 Vintage Synths Sliding Puzzle 🧩

Welcome to the **Vintage Synths Sliding Puzzle**, a gorgeous, retro-futuristic sliding tile game featuring beautifully detailed digital renderings of classic analog and digital synthesizers. 

This app was built as a fun, tactile, and highly responsive **fidget toy** designed to play with, unwind, and celebrate the legendary instruments that shaped electronic music history.

## 🕹️ Features

* **Classic Synths:** Match panels for iconic machines including:
  * 🎛️ **Minimoog**
  * 🍊 **ARP Odyssey Mk III**
  * ⚡ **Yamaha DX7**
  * 🔴 **Roland Juno-60**
  * 🌀 **Roland TB-303**
  * 🥁 **Roland TR-808 / TR-909**
  * 🎹 **Casio CZ-1**
  * 🎚️ **Akai MPC60**
  * 💾 **E-mu SP-1200**
* **Brand-Specific UI:** The header changes colors and typography dynamically to match the exact aesthetic and branding of the active synth.
* **Physics & Feel:** Responsive drag-and-touch physics let you flick tiles smoothly.
* **Solvers & Previews:** Stuck? Use the auto-solve helper or peek at the preview of the completed image.
* **Live Stats:** Track your moves and elapsed time in real-time during your active game session. No data is tracked, saved, or sent anywhere else—everything runs entirely locally on your device

## 🛠️ Technical Details

This project is built using a modern, lightweight, and performance-oriented web stack:

* **Core Framework:** [Lit](https://lit.dev/) (v3) - leveraging fast, native Web Components with minimum overhead.
* **Language:** [TypeScript](https://www.typescriptlang.org/) for robust types and reliable component state management.
* **Build Tool:** [Vite](https://vite.dev/) for extremely fast development cycles and optimized, dependency-free production bundles.
* **Styling:** Custom Vanilla CSS with scoped `:host` styles, custom CSS properties for dynamic branding, retro scanline/screen overlays, and responsive flexbox layouts.

### Getting Started

To run the project locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Build the production application (outputs to `/docs` for GitHub Pages hosting):
   ```bash
   npm run build
   ```
